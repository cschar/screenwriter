# DRFScreenwriter 

experimenting w/ drf + react for making mini screenwriter stories

## developing

console 1
> pip install -r requirements.txt
> ./manage.py runserver 3000

console 2
> cd reactfrontend && npm install
> npm run start

## Building frontend 

```
export API_HOSTNAME='https://api.mydomain.com'
cd reactfrontend
npm run build
```
